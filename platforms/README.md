## Request a Platform

Don't see your tool? [Open an issue](https://github.com/makerprism/feedmansion-ai-agents/issues) and we'll add it.

## Contributing

Have a configuration for another platform? PRs welcome!

1. Fork the repo
2. Add your platform folder to `platforms/`
3. Include a README.md with setup instructions
4. Submit a pull request
